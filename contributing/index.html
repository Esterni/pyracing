    Contributing (Under Construction)¶

    Please feel free to open issues, offer suggestions, or fork and submit a pull request. We only ask that you be willing to openly discuss things in a respectful manner. The contributors of this project use Discord as the primary means of communication; The iRacing Open Wheel server was created by the author of this project and hosts the channels for discussion there. When joining, please ask Jacob Anderson for the role to see the appropriate channels.

    Code Style¶

    This Repo follows the PEP8 Styleguide.

    • Line lengths are 79 characters.
    • Variable names use snake_case.
    • Class names use CamelCase. (CapitalizedWords)
    • Constants use ALL_CAPS.
